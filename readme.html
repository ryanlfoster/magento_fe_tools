Magento Front-end Starter Kit

v.1.0.0

Be sure that on your pc installed next software and utilities:

  1. NodeJs
  2. RubyInstaller
  3. Compass
  4. Grunt Task runner

Getting started

Copy starter kit to your magento project


        app
        |-- design
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- layout
        |------------- local.xml

        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- scss
        |------------- styles.scss
        |----------- css
        |------------- styles.css
        |----------- fonts
        |----------- gui
        |------------- index.html
        |----------- images
        |------------- uncompressed
        |----------- svg
        |------------- uncompressed
        |----------- js
        |------------- app
        |--------------- main.js
        |------------- plugins
        |--------------- plugin-1.jquery.js
        |--------------- plugin-2.jquery.js
        |--------------- plugin-1.jquery.js
        |------------- vendor
        |--------------- jquery-1.11.1.min.js
        |------------- plugins.min.js
        |
        |----------- .jshintrc      // JsHint Rules
        |----------- config.rb      // Compass config
        |----------- Gruntfile.js   // Grunt Task Runner
        |----------- package.json   // Grunt plugins dependencies

Download Grunt dependencies

Open in terminal your skin folder and type command 
$ npm install

this command download all dependencies which described in the file package.json
into folder node_modules

Starter kit contains next Grunt plugins:

  1. grunt-contrib-watch - watched files changes
  2. grunt-contrib-imagemin - Minify images
  3. grunt-contrib-imagemin - Minify SVG files
  4. grunt-contrib-uglify - Minify js files with UglifyJS.

For more plugins look at Grunt plugins

Run Grunt utils

1. Compress images

Put images into folder


        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- images
        |------------- uncompressed
        |--------------- image1.png
        |--------------- image2.jpg
        |--------------- image3.gif
and run command 
$ grunt imagemin
this command create compressed images into folder 

        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- images
        |------------- uncompressed
        |--------------- image1.png
        |--------------- image2.jpg
        |--------------- image3.gif
       +|------------- image1.png // compressed
       +|------------- image2.jpg // compressed
       +|------------- image3.gif // compressed

2. Compress svg files

Put images into folder


        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- svg
        |------------- uncompressed
        |--------------- svg-icon-1.png
        |--------------- svg-icon-2.jpg
        |--------------- svg-icon-3.gif
and run command 
$ grunt svgmin
this command create compressed svg files into folder 

        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- svg
        |------------- uncompressed
        |--------------- svg-icon-1.svg
        |--------------- svg-icon-2.svg
        |--------------- svg-icon-3.svg
       +|------------- svg-icon-1.svg // compressed
       +|------------- svg-icon-2.svg // compressed
       +|------------- svg-icon-3.svg // compressed

2. Concat and compressed all jquery plugins into one file

$ grunt uglify

Get all files from


        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- js
        |------------- plugins
        |--------------- plugin-1.jquery.js
        |--------------- plugin-2.jquery.js
        |--------------- plugin-3.jquery.js
and convert it into one compressed file 

        skin
        |----- frontend
        |------- <package>
        |--------- <theme>
        |----------- js
       +|------------- plugins.min.js // compressed
